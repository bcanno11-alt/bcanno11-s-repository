What is Hammer?

Hammer is a level editor used for games which are made on the Source Engine. These are games like:

  • Half-Life 2
  • Team Fortress 2
  • Counter-Strike: Source
  • Garrys Mod
  • Left 4 Dead

I mainly use it for Garrys Mod.

Controls/Hotkeys

Key Function
[, ] Resize grid
Scroll Zoom
Shift+Right Click Move Camera in 3D window
WASD Move around in 3D window
Mousewheel click+Drag Move around in front/side/top window
Shift+Drag brush Duplicates selected brush to dragged position
